.SH DESCRIPTION
.PP
Secure Personal Cloud is a file storage and synchronization service developed in CS251 Project in IIT Bombay.
SPC allows users to store files on their servers, synchronize files across devices, and share files.
Unlike Google drive it also has an encryption schema decided by each user so as to prevent hacking.
.PP
Mandatory arguments to long options are mandatory for short options too.
.TP
\fBdownload\fR
Downloads directories/files from server if 
already login otherwise asks username and 
password
.TP
\fBen\-de\fR
Info about encryption and decryption schema
.TP
\fBen\-de\fR \fBlist\fR
Lists all the supported encryption schema
.TP
\fBen\-de\fR \fBupdate\fR
To update the default schema and keys manually
or by providing a file as an argument
e.g., "spc en\-de update <file\-path>"
.TP
\fBen\-de\fR \fBdump\fR
To create a file for the already in place schema
e.g., "spc en\-de dump <file\-path>"
.TP
\fB\-\-help\fR, \fBhelp\fR
Displays help
.TP
\fBremember\fR
Remembers the password untill logout
.TP
\fB\-lo\fR, \fBlogout\fR
To logout the session in linux client
.TP
\fBserver\fR
Displays server's ip and port number
.TP
\fBserver\fR \fBset\-url\fR
To set the server's url e.g.,
"spc server set\-url 10.196.2.33" sets server url
to 10.196.2.33
.TP
\fBstatus\fR
Displays the similarities and differences between
files in the client and the server
.TP
\fBupload\fR
Uploads directories/files provided as aruguments
if already login otherwise asks username and
password
.TP
\fB\-\-version\fR
Displays the version
